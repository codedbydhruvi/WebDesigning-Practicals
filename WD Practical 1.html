Understanding the Structure of an HTML5 Document

About Semantic Elements

Semantic elements in HTML5 clearly describe their meaning to both the browser and the developer. They improve code readability, accessibility, and SEO.

Header Element

The <header> element represents introductory content or navigation links for a webpage or section.

Section Element

The <section> element defines a section in a document, such as chapters, headers, or thematic grouping of content.

Article Element

The <article> element represents independent and self-contained content such as blog posts, news articles, or forum posts.
